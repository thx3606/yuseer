# نظام إدارة مدارس تحفيظ القرآن الكريم والمتون العلمية
## Quranic Schools Management System

---

## 📖 نظرة عامة | Overview

نظام شامل ومتكامل لإدارة مدارس تحفيظ القرآن الكريم، المتون العلمية، والقاعدة النورانية بشكل احترافي وذكي.

**A comprehensive and integrated system for managing Quranic memorization schools, Islamic texts (Mutoon), and Noorani Qaida with professional and intelligent features.**

---

## 🏗️ البنية التقنية | Technology Stack

### Backend
- **Node.js** + **Express.js** - خادم التطبيق
- **PostgreSQL** - قاعدة البيانات الرئيسية
- **Prisma ORM** - إدارة قاعدة البيانات
- **Redis** - التخزين المؤقت
- **Socket.io** - الرسائل الفورية

### Frontend
- **React.js** + **Next.js 14** - واجهة المستخدم
- **TypeScript** - لغة البرمجة
- **Tailwind CSS** - التصميم
- **Shadcn/ui** - مكونات واجهة المستخدم
- **React Query** - إدارة البيانات

### Authentication & Security
- **JWT** - المصادقة
- **bcrypt** - تشفير كلمات المرور
- **RBAC** - نظام الصلاحيات

### Integrations
- **Nodemailer** - البريد الإلكتروني
- **Twilio / WhatsApp Business API** - واتساب
- **PDFKit** - إنشاء الشهادات
- **ExcelJS** - تصدير البيانات

## 🚀 البدء | Getting Started

### المتطلبات | Prerequisites
- Node.js v18+
- PostgreSQL v14+
- Redis v6+
- npm أو yarn

### التثبيت | Installation

```bash
# استنساخ المشروع
git clone [repository-url]

# تثبيت التبعيات - Backend
cd backend
npm install

# تثبيت التبعيات - Frontend
cd ../frontend
npm install

# إعداد قاعدة البيانات
cd ../backend
npx prisma migrate dev
npx prisma generate

# تشغيل البيئة التطويرية
npm run dev
```
