## 🚀 خطوات التشغيل السريعة

```bash
# 1. تثبيت التبعيات
cd backend && npm install

# 2. إعداد البيئة
cp .env.example .env
# عدّل DATABASE_URL وباقي الإعدادات

# 3. إنشاء قاعدة البيانات
createdb quran_schools

# 4. تشغيل Migration
npx prisma migrate dev
npx prisma generate

# 5. تشغيل الخادم
npm run dev
```

الخادم يعمل على: **http://localhost:5000**

## 🎨 التقنيات المستخدمة

### Backend Stack
```
- Runtime: Node.js v18+
- Framework: Express.js
- Language: TypeScript
- Database: PostgreSQL
- ORM: Prisma
- Auth: JWT + Passport.js
- Encryption: bcrypt
- Email: Nodemailer
- WhatsApp: Twilio
- PDF: PDFKit
- Excel: ExcelJS
- Logging: Winston
- Validation: express-validator
```

### Security
```
- Helmet.js (HTTP Headers)
- CORS (Cross-Origin)
- Rate Limiting
- JWT Authentication
- Role-Based Access Control (RBAC)
- Password Hashing (bcrypt)
- Input Validation
```
